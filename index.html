معرفی برنامه

Minecraft Pixel Editor یک ابزار حرفه‌ای برای طراحی مدل‌های پیکسلی Minecraft است. با این برنامه می‌توانید بلوک‌ها را به راحتی طراحی کرده و مدل‌های خود را با لایه‌های مختلف و قابلیت Undo/Redo مدیریت کنید.

نصب و پیش‌نیازها

  • نصب Python 3
  • نصب کتابخانه PIL (Pillow) با دستور زیر: 
    pip install pillow
  • دانلود فایل minecraft_pixel_editor.py

اجرای برنامه:

python minecraft_pixel_editor.py

ویژگی‌های اصلی

  • چند لایه با قابلیت روشن/خاموش و نام‌گذاری
  • Undo و Redo نامحدود
  • پالت کامل رنگ‌های Minecraft + رنگ دلخواه
  • Brush قابل تنظیم برای نقاشی روی چند پیکسل همزمان
  • بزرگ‌نمایی/کوچک‌نمایی دقیق
  • ذخیره مدل نهایی با کیفیت بالا و ذخیره هر لایه جداگانه
  • بارگذاری تصویر مرجع برای طراحی دقیق

راهنمای استفاده

انتخاب اندازه شبکه

پس از اجرای برنامه، اندازه شبکه دلخواه خود را وارد کنید (مثلاً 16x16).

انتخاب رنگ و نقاشی

می‌توانید رنگ‌ها را از پالت بلوک‌های Minecraft انتخاب کنید یا رنگ دلخواه خود را تعیین کنید. برای نقاشی، کافی است روی Canvas کلیک کرده یا بکشید. Brush قابل تنظیم است.

مدیریت لایه‌ها

  • افزودن لایه جدید
  • انتخاب لایه برای طراحی
  • روشن/خاموش کردن لایه‌ها برای مشاهده ترکیبی

Undo / Redo

هر تغییر روی لایه‌ها قابل بازگشت است و می‌توانید به راحتی عملیات Undo و Redo انجام دهید.

Bigger Brush و Zoom

Brush را برای نقاشی روی چند پیکسل همزمان می‌توانید تغییر دهید و با Zoom Canvas را بزرگ یا کوچک کنید.

ذخیره مدل

  • ذخیره کل تصویر نهایی
  • ذخیره هر لایه به صورت جداگانه
